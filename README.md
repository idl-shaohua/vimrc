# Vimrc
This is the collection of useful plug-in for vim.
 - scrooloose/nerdtree
 - scrooloose/nerdcommenter
 - bling/vim-airline
 - taglist.vim

### Pre-requirements
 - vim
 - git
 - ctags
 - powerline fonts (for vim-airline)

### Installation
```
$ sh -c "$(wget https://raw.githubusercontent.com/idl-shaohua/vimrc/master/install.sh -O -)"
```
